# Ishtar

Ishtar Website Front End Implementation.

## Installation 

  1. Clone this Repo
  2. Navigate to the Repo
  3. run `npm install`
  4. run `ng serve --o`
  5. Voila The Application is Running :)
  
## About

  1. Angular NGX-UI Bootstrap Style Tabs
  2. Images Overlay
  3. Animations, Zoom and Reveal
